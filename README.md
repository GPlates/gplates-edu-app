# GPlates Education Ionic App

* nvm use 16
* source ./set-env.sh
* ionic capacitor add android
* ionic capacitor run android
* ionic capacitor run ios
* npx cap open android
* npx cap serve
* npx cap open ios

#### use chrome inspect to debug android chrome://inspect/#devices
